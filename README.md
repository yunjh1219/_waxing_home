## 프로젝트 소개
이 프로젝트는 피플왁싱(가칭)을 위한 실시간 예약 및 정보 제공 웹사이트입니다.
고객이 피플왁싱에 대한 정보를 쉽게 확인하고 간편하게 예약할 수 있도록 하며,
관리자가 예약 및 고객 정보를 효율적으로 관리할 수 있도록 설계되었습니다.

## 주요 기능
[고객 기능]
- 피플왁싱 서비스 정보 제공
- 고객 예약 시스템
- 고객 후기 및 리뷰
- 가게 정보 제공(공지, 이벤트 등) 

[관리자 기능]
- 회원 및 예약 관리
- 콘텐츠 관리(공지, 이벤트 등)

## 기술 스택
- 프론트엔드: HTML5, CSS3, JavaScript, Thymeleaf
- 백엔드: Java, Spring Boot, JPA
- 데이터베이스: MySQL
- 개발 도구: IntelliJ, Git
